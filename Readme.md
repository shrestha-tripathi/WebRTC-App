# WebRTC Video Application

## Features

- 1 on 1 Video Call
- Video Conferencing by creating rooms (WIP)

## Project Init
```bash
git clone https://github.com/shrestha-tripathi/WebRTC-App.git
cd WebRTC-App
cd server
npm i
cd ../client
npm i
```

## Steps to Run the server
```bash
cd server
npm run dev
```

## Steps to Run the client
```bash
cd client
npm run start
```
